Plataforma de Gestión de Datos Clínicos
Este proyecto es una aplicación web diseñada para la administración de información médica y de investigación. Su objetivo principal es permitir la gestión de pacientes, usuarios y la recolección de datos a través de formularios dinámicos.

Funcionalidades Principales
Gestión de Usuarios: Registro, autenticación y control de roles y permisos.

Administración de Pacientes: Registro y seguimiento de fichas de pacientes.

Formularios Dinámicos: Creación y respuesta de encuestas o formularios personalizados (CRF).

Reportes y Exportación: Generación de documentos PDF y exportación de datos a Excel.

Estadísticas: Visualización de gráficos y métricas sobre los datos ingresados.

Auditoría: Registro de actividades y cambios en el sistema.

Tecnologías Utilizadas
Backend:

Java

Spring Boot (Seguridad, JPA, Web, Mail)

MySQL (Base de datos)

Librerías para PDF y Excel (OpenHTMLtoPDF, Apache POI)

Frontend:

React 

Vite

Bootstrap 

Infraestructura:

Docker y Docker Compose (para facilitar el despliegue)
